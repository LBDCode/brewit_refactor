Parameter Required Type Description
key yes* string Your brewIt API key (ex: 'key=233as-234-dc-2341-3').
s yes* string Search query text (ex: 's=brown+ale'). Every call must have either s or r parameters. If both are included, the query will default to the search parameter, and ignore the random command.
r yes* integer Random response generator based on number of desired recipes (ex: 'r=24'). The upper limit for r is 50 results. Every call must have either s or r parameters. If both are included, the query will default to the search parameter, and ignore the random command.
type no string Filter results by style of beer (ex: 'type=ipa').
low-ibu and high-ibu no float Filter results by low or high IBU limits; these act as >= and <= operators, and can be used in combination (ex: 'low-ibu=30' will return recipes with IBU >= 30. 'low-ibu=30&high-ibu=60' will return recipes with IBU >=30 and <=60').
low-abv and high-abv no float Filter results by low or high ABV limits; these act as >= and <= operators, and can be used in combination (ex: 'low-abv=4' will return recipes with ABV >= 4. 'low-abv=4&high-abv=8' will return recipes with IBU >=4 and <=8').
limit no integer Restrict the number of results (ex: 'limit=30'). The maximum number of results is 50, use offset to paginate searches.
offset no integer Use with the a user defined limit or default limit (50 results) to paginate results (ex: 'offset=20' will return results 21-70 of the search, while 'limit=30&offset=10' will return results 11-40 of the search)
